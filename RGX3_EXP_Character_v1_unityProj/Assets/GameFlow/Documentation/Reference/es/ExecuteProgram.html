Execute Program

Inicia la ejecucion del programa (Program) especificado.

Parámetros:

  • Program: El programa (Program) a ejecutar.
  • Force Restart: Indica si el programa debe ser reiniciado antes de comenzar a ejecutarlo.
  • Wait For Completion: Indica si la ejecución del programa actual debe quedarse a la espera de que el programa ejecutado finalice.
